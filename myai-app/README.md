# 🎭 MyAI-RolePlay v5.9.2

> AI 角色扮演沉浸式体验 — 纯前端 PWA，直连 DeepSeek API

**在线体验：** [https://myai-app-eight.vercel.app](https://myai-app-eight.vercel.app)

## ✨ 功能特性

### 🎭 角色系统
- **自定义角色** — 名称、性格、System Prompt、语气风格全自定义
- **✨ AI 一句话生成角色** — 输入一句描述，AI 自动填充全部人设字段
- **角色深度字段** — 世界观设定、外貌、说话风格、关系、秘密
- **头像选择** — 点击拍照/从相册选择，或粘贴 URL
- **对话规则 / 禁区** — 设定角色不能聊的话题
- **剧情摘要** — 手动 + 自动摘要，AI 始终记住重要剧情
- **预设角色** — 内置角色模板，开箱即用

### 💬 单聊模式
- **流式响应** — DeepSeek V3/R1 实时流式输出
- **双层响应解析** — 自动提取 `<think>` 推理 + `<inner>` 内心戏
- **😊 表情头像** — AI 回复自动识别情绪，头像实时变化（8 种表情）
- **输出长度控制** — 短 / 标准 / 长 三档，全局统一控制
- **消息管理** — 删除、重新生成、记忆固定
- **对话分支 (Fork)** — 在任意消息处分叉，探索不同剧情走向
- **自动摘要** — 20 条触发，保留完整历史 + 摘要上下文
- **TTS 语音朗读** — 支持自动朗读和声线选择
- **🪙 Token 消耗量显示** — 每条 AI 回复显示精确 Token 用量

### 👥 群聊模式
- **多角色群聊** — 2-8 个 AI 角色同时对话
- **导演模式** — 用户作为导演引导话题
- **🎬 AI 盲盒事件** — 一键生成 AI 世界事件，无需手动输入
- **影子导演** — AI 自动检测剧情卡壳并注入情境事件
- **悄悄话** — 私聊某个角色，目标角色立即回应
- **PASS 机制** — 无关角色自动跳过，对话更自然
- **⚡ 命令菜单** — 向上弹出的 Popover，收纳事件/悄悄话/关系雷达/指定角色发言
- **▶️ 继续按钮** — 紫色渐变发光 FAB，一键继续一轮
- **剧本基调** — 搞笑日常 / 校园恋爱 / 悬疑推理等风格约束
- **📊 关系矩阵** — 角色间好感度(-100~100)，AI 每轮自动分析更新
- **🔗 动态关系标签** — ally/rival/crush/mentor/enemy 等动态关系，AI 自动提取
- **🤫 共同秘密** — 角色间自动记忆共享秘密，影响互动细节
- **雷达图可视化** — Chart.js 雷达图 + 视角切换 + 手动 Slider 微调
- **🔗 跨场景记忆同步** — 群聊重要事件自动写入角色永久记忆
- **💭 内心独白** — 群聊也支持角色内心戏思想气泡
- **🪙 Token 消耗量** — 群聊每条消息显示 Token 用量

### 📅 剧情时间线
- **自动事件提取** — AI 后台分析对话，提取关键剧情事件（承诺、冲突、转折等）
- **重要度分级** — ⚡ 高 / 📌 中 / · 低，三级重要性标记
- **Prompt 注入** — 时间线事件自动注入上下文，AI 不会忘记关键剧情
- **单聊 + 群聊通用** — 两种模式均支持独立时间线
- **可视化管理** — 📅 时间线 Tab 查看/删除事件
- **锁冲突重试** — 时间线与摘要互斥锁冲突时自动 10 秒重试

### 📖 对话导出为故事
- **直接导出** — 一键将对话转换为精美排版的 HTML 故事文档
- **AI 润色导出** — AI 将对话改写为通顺的文学叙事，配 SVG 圆环倒计时
- **专业排版** — 银蓝动作描写 + 暖奶油正文 + 衬线字体，阅读体验优雅

### 📔 日记系统
- **角色日记** — AI 以角色视角撰写日记，记录当天情感与事件
- **日记链** — 新日记自动延续旧日记中的目标与情绪
- **一键开启新一天** — 🌙 菜单开启下一天，角色以新状态回应
- **历史日记浏览** — 按角色/群聊过滤，沉浸式翻页阅读

### 👤 全局用户画像
- **自动特征提取** — AI 后台分析聊天记录，提取用户偏好、性格、事实
- **五大类别标签** — 偏好 / 性格 / 事实 / 风格 / 边界，彩色标签可视化
- **Prompt 自动注入** — 用户画像融入每次对话，AI 更懂你
- **手动管理** — 设置面板中可添加/删除/查看所有标签

### 🧠 智能记忆管理
- **📌 永久记忆** — 钉选重要消息或手动创建，AI 始终记住
- **🔗 群聊记忆同步** — 每 6 轮群聊自动提取重要事件，写入角色记忆
- **🗜️ 智能压缩** — 记忆超过 20 条时自动压缩最旧的 10 条为精炼摘要
- **AI 精简** — 一键让 AI 将冗长记忆精简为 50 字事实陈述
- **来源标识** — 🔗 群聊同步 / 📝 手动创建 / 🎭 消息钉选 / 📦 历史摘要
- **历史消息保留** — 自动摘要不再删除旧消息，用户可回看完整聊天记录

### 💾 数据备份与恢复
- **完整数据备份** — 一键导出全部数据（设置、角色、群聊、日记、用户画像）
- **文件选择器恢复** — 支持文件选择和拖放导入，导入前可预览内容
- **数据验证** — 导入时检查数据完整性，防止损坏数据覆盖

- **🎓 新手引导 Overlay** — 3 步引导（选角色 → 聊天 → 群聊），首次启动展示，含进阶提示
- **🕐 回访欢迎横幅** — "已经过去 X 天了，XX 一直在想你"，情感留存钩子

### 🛡️ 安全与稳定
- **发送锁** — 防止快速点击重复发送
- **竞态保护** — 角色切换时中止旧请求
- **流式守卫** — 流式响应期间禁止重新生成/删除/新一天等并发操作
- **超时保护** — 30 秒标准 / 120 秒 Reasoner 动态超时

- **输入验证** — XSS 防护、DOMPurify 清洗、URL 安全校验
- **localStorage 持久化** — 所有数据本地存储，刷新不丢失

## 🛠️ 技术栈

- **前端框架：** Vue 3 (Composition API + `<script setup>`)
- **构建工具：** Vite 7
- **样式：** Tailwind CSS 4 + 原生 CSS (Glassmorphism)
- **图表：** Chart.js + vue-chartjs (关系雷达图)
- **Markdown：** marked + DOMPurify
- **测试：** Vitest + jsdom (206 个测试用例，11 个测试文件)
- **部署：** Vercel (Git 推送自动部署)
- **AI 模型：** DeepSeek V3 / R1（兼容 OpenAI 格式 API）

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/LiskaDev/myai-app.git
cd myai-app

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 运行测试
npx vitest run

# 构建生产版本
npx vite build
```

## 📦 部署到 Vercel

```bash
cd myai-app && npx vite build && cd .. && npx vercel --prod --yes
```

> ⚠️ 从仓库根目录 (`MyAI_RolePlay/`) 运行，Vercel Root Directory 设置为 `myai-app`
